# 🏔️ 中国雪山图鉴

一个展示中国十二座著名雪山的静态网站，包含雪山档案、知识库、海拔数据可视化等功能。

## 功能

- **首页** — 轮播大图 + ECharts 海拔柱状图与省份分布环形图、雪山数据对比表格
- **雪山档案** — 12 座雪山图文卡片，可按省份/难度筛选，点击查看详情
- **知识库** — 雪山知识翻转卡片，分类浏览（地质成因、登山装备、登山历史等）
- **加入我们** — 关于页 / 联系信息

## 技术栈

- 原生 HTML + CSS + JavaScript
- ECharts 数据可视化
- 部署于 GitHub Pages

## 本地运行

直接双击 `index.html` 打开，或用 VS Code 的 Live Server 启动。

## 说明

仅供课程学习使用，图片来源于 Unsplash 等开放图库。
